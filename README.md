# RelaisCtl

This is a private IOT project, where this service controls a Raspberry Pi Relais/Relay
(["Zero Relay"](https://github.com/sbcshop/Zero-Relay)) via custom MQTT triggers.
The main use case is to control the relays from [X-Sense smoke detectors](https://www.x-sense.com/).
The [Home Assistant MQTT integration](https://github.com/cschnidr/openhab-xsense-mqtt) of X-Sense is used for this.

More MQTT triggers might be added in the future. External MQTT broker required.
